# Windows-ServerLab-Dokumentation (Work in Progress)

In diesem Repository dokumentiere ich die Einrichtung und Verwaltung einer virtuellen Windows Serverumgebung inkl. DomainController, DNS; Active Directory und Core-Server für SQL Datenbank.

## Ziel

Ziel des Systems ist, eine möglichst praxisnahe Umgebung zu schaffen, um realitätsnahe Szenarios zur Kompetenzbildung zu simulieren. Bereits erworbene Fähigkeiten sollen geschärft, ergänzt und erweitert werden.
